# 设计系统：浅色科技风格

## 1. 视觉主题与氛围

浅色科技风是一种被控制的戏剧感——纯粹的黑色与近乎白色的广阔空间作为电影般的背景，内容如同画廊中的雕塑般被呈现。设计哲学极度精简：每个像素都服务于内容本身，界面自身退隐直至无形。这不是作为审美偏好的极简主义，而是作为对内容致敬的极简主义。

字体系列采用光学尺寸调整，根据字号大小自动调整字母形态。在展示尺寸（35px）下，字重 600 搭配紧凑行高 1.07 和微妙的负字间距（-0.28px），创造出仿佛精密机械加工而非排版而成的标题——精确、自信、毫不道歉般地直接。在正文尺寸（19px）下，字间距略微放松（-0.374px），行高打开到 1.47，营造出舒适但从不松懈的阅读节奏。

色彩故事是二元分明的。内容区域在纯黑（`#000000`）背景配白色文字与浅灰（`#f5f5f7`）背景配近黑文字（`#1d1d1f`）之间交替。这创造了电影般的节奏——深色区域感觉沉浸且高级，浅色区域感觉开阔且信息丰富。唯一的彩色 accent 是科技蓝（`#0071e3`），专门保留给交互元素：链接、按钮和聚焦状态。这一单一的 accent 色在中性色的海洋中让每个可点击元素都具有无可置疑的可见性。

**关键特征：**

- 光学尺寸调整字体系列——字母形态根据尺寸语境自动适应
- 二元明暗区域节奏：黑色（`#000000`）交替浅灰（`#f5f5f7`）
- 单一 accent 色：科技蓝（`#0071e3`）专门保留给交互元素
- 产品即Hero的摄影风格，纯色背景——无渐变、无纹理、无干扰
- 极度紧凑的标题行高（1.07-1.14）创造出压缩的、广告牌般的影响力
- 全宽区域布局，居中内容——画布充满整个视野
- 药丸形 CTA（980px 圆角）创造柔和、亲切的操作按钮
- 区域之间 generous 的留白，让每个内容时刻自由呼吸

## 2. 色彩调色板与角色

### 主色

- **纯黑**（`#000000`）：Hero区背景，沉浸式展示。最暗的画布衬托最亮的内容。
- **浅灰**（`#f5f5f7`）：交替区域背景，信息区域。不是纯白——轻微的蓝灰色调防止冰冷感。
- **近黑**（`#1d1d1f`）：浅色背景上的主要文字，深色按钮填充色。比纯黑略暖，阅读更舒适。

### 交互色

- **科技蓝**（`#0071e3`）：主 CTA 背景，聚焦环。设计中唯一的彩色。
- **链接蓝**（`#0066cc`）：行内文字链接。比科技蓝略深，文字可读性更好。
- **亮蓝**（`#2997ff`）：深色背景上的链接。更高亮度以在黑色区域形成对比。

### 文字色

- **白色**（`#ffffff`）：深色背景上的文字，蓝色/深色 CTA 上的按钮文字。
- **近黑**（`#1d1d1f`）：浅色背景上的主要正文。
- **黑色 80%**（`rgba(0, 0, 0, 0.8)`）：次要文字。略微柔和。
- **黑色 48%**（`rgba(0, 0, 0, 0.48)`）：三级文字，禁用状态。

### 表面与深色变体

- **深色表面 1**（`#272729`）：深色区域中的卡片背景。
- **深色表面 2**（`#262628`）：深色语境中的微妙表面变化。
- **深色表面 3**（`#28282a`）：深色背景上的悬浮卡片。
- **深色表面 4**（`#2a2a2d`）：最高深色表面高度。
- **深色表面 5**（`#242426`）：最深深色表面色调。

### 按钮状态

- **按钮激活**（`#ededf2`）：浅色按钮的激活/按下状态。
- **按钮默认浅色**（`#fafafc`）：搜索/筛选按钮背景。
- **遮罩**（`rgba(210, 210, 215, 0.64)`）：媒体控件遮罩，叠加层。

### 阴影

- **卡片阴影**（`rgba(0, 0, 0, 0.22) 3px 5px 30px 0px`）：产品卡的柔和弥散阴影。偏移量和宽模糊创造出自然、摄影般的阴影效果。

### 原则

- **光学尺寸作为哲学核心**：字体系列自动在 Noto Sans SC 和 Noto Sans SC 光学尺寸之间切换。Noto Sans SC 版本有更宽的字间距和更细的笔画，针对大尺寸优化；Noto Sans SC 版本更紧凑、更坚固，针对小尺寸优化。
- **字重克制**：字重范围从 300（细）到 700（粗），但大多数文字停留在 400（常规）和 600（半粗）。300 字重仅出现在大型装饰文字上。700 字重很少使用，仅用于粗体卡片标题。
- **全尺寸负字间距**：与大多数仅在标题上调整字间距的系统不同，本系统在正文尺寸也应用微妙的负字间距（19px 时 -0.374px，16px 时 -0.224px，12px 时 -0.12px）。这创造了普遍紧凑高效的文字。
- **极端行高范围**：标题压缩到 1.07，正文打开到 1.47，某些按钮语境拉伸到 2.41。这种戏剧性范围仅通过节奏就创造了清晰的视觉层级。

## 4. 组件样式

### 按钮

**主蓝色（CTA）**

- 背景：`#0071e3`（科技蓝）
- 文字：`#ffffff`
- 内边距：8px 15px
- 圆角：8px
- 边框：1px solid transparent
- 字体：16px，字重 400
- 悬停：背景略微变亮
- 激活：`#ededf2` 背景偏移
- 聚焦：`2px solid var(--sk-focus-color, #0071E3)` 轮廓
- 用途：主行动号召（"购买"、"立即体验"）

**主深色**

- 背景：`#1d1d1f`
- 文字：`#ffffff`
- 内边距：8px 15px
- 圆角：8px
- 字体：16px，字重 400
- 用途：次要 CTA，深色变体

**药丸形链接（了解更多 / 选购）**

- 背景：transparent
- 文字：`#0066cc`（浅色 bg）或 `#2997ff`（深色 bg）
- 圆角：980px（完整药丸形）
- 边框：1px solid `#0066cc`
- 字体：16px
- 悬停：下划线装饰
- 用途："了解更多"和"选购"链接——标志性内联 CTA

**筛选 / 搜索按钮**

- 背景：`#fafafc`
- 文字：`rgba(0, 0, 0, 0.8)`
- 内边距：0px 14px
- 圆角：11px
- 边框：3px solid `rgba(0, 0, 0, 0.04)`
- 聚焦：`2px solid var(--sk-focus-color, #0071E3)` 轮廓

### 卡片与容器

- 背景：`#f5f5f7`（浅色）或 `#272729`-`#2a2a2d`（深色）
- 边框：无（边框很少使用）
- 圆角：5px-8px
- 阴影：`rgba(0, 0, 0, 0.22) 3px 5px 30px 0px` 用于悬浮产品卡
- 内容：居中， generous 内边距
- 悬停：无标准悬停状态——卡片静态，内部链接可交互

### 图片处理

- 产品在纯色背景上（黑色或白色）——无背景、无语境、只有物体本身
- 全出血区域图片跨越整个宽度
- 产品摄影极高分辨率，带微妙阴影
- 生活方式图片限制在圆角容器中（12px+ 圆角）

### 特色组件

**产品Hero模块**

- 全宽度区域，纯色背景（黑色或 `#f5f5f7`）
- 产品名称作为主标题（Noto Sans SC 字体，35px，字重 600）
- 一行描述， lighter 字重
- 并排两个药丸形 CTA："了解更多"（描边）和"购买"/"立即体验"（填充）

**产品网格磁贴**

- 正方形或近似正方形卡片，对比背景
- 产品图片占据磁贴 60-70%
- 产品名称 + 一行描述在下方
- 底部"了解更多"链接对

**功能对比条**

- 水平排列的产品/方案变体
- 每个变体作为垂直卡片，包含图片、名称和关键指标
- 极简装饰——内容自己说话

## 5. 布局原则

### 间距系统

- 基本单位：8px
- 刻度：2px, 4px, 5px, 6px, 7px, 8px, 9px, 10px, 11px, 14px, 15px, 17px, 20px, 24px
- 显著特征：刻度在小尺寸（2-11px）密集，1px 增量，然后跳转为更大的步长。这允许对排版和图标对齐进行精确的微调。

### 网格与容器

- 最大内容宽度：约 980px（药丸按钮中反复出现的"980px 圆角"呼应了这个宽度）
- Hero区：全宽度区域，居中内容块
- 产品网格：居中容器内的 2-3 列布局
- Hero时刻单列——一个产品，一条信息，全神贯注
- 无可见网格线或排水沟——间距创造隐含结构

### 留白理念

- **电影般的呼吸空间**：每个内容区域占据全视口高度（或接近）。内容之间的留白不是空的——它是场景中之间的暂停。
- **通过色块实现垂直节奏**：不使用仅靠间距分隔区域，而是使用交替背景色（黑色、`#f5f5f7`、白色）。每次颜色变化都标志着新的"场景"。
- **内部压缩，外部扩展**：文字块紧密设置（负字间距，紧凑行高），而它们周围的空间是广阔的。这在密度和开放之间创造了张力。

### 圆角刻度

- 微型（5px）：小型容器，标签
- 标准（8px）：按钮，产品卡，图片容器
- 舒适（11px）：搜索输入，筛选按钮
- 大（12px）：功能区面板，生活方式图片容器
- 完整药丸形（980px）：CTA 链接（"了解更多"），导航药丸
- 圆形（50%）：媒体控件（播放/暂停，箭头）

## 6. 深度与层次

| 级别             | 处理方式                               | 用途                 |
| ---------------- | -------------------------------------- | -------------------- |
| 平面（0 级）     | 无阴影，纯色背景                       | 标准内容区域，文字块 |
| 微妙悬浮（1 级） | `rgba(0, 0, 0, 0.22) 3px 5px 30px 0px` | 产品卡，浮动元素     |
| 聚焦（无障碍）   | `2px solid #0071e3` 轮廓               | 所有交互元素的聚焦   |

**阴影理念**：阴影使用极其节制。主阴影（`3px 5px 30px`，0.22 透明度）柔和、宽泛、偏移——模仿漫射工作室灯光在物理物体下方投射的自然阴影。这强化了"内容即物理雕塑"的隐喻。大多数元素根本没有阴影；高度来自背景色对比（深色卡片在更深的背景上，或浅色卡片在稍有不同的灰色上）。

### 装饰深度

- 区域颜色过渡：深度由黑色和浅灰区域之间的交替暗示，而非由阴影暗示
- 产品摄影阴影：内容本身在摄影中投射阴影，因此不需要添加合成阴影

## 7. 该做与不该做

### 该做

- 在 20px 及以上使用 Noto Sans SC 字体，20px 以下使用 Noto Sans SC 字体——尊重大小边界
- 在所有文字尺寸应用负字间距（不仅是标题）——全局紧凑
- 仅将科技蓝（`#0071e3`）用于交互元素——它必须是唯一的 accent
- 在黑色和浅灰（`#f5f5f7`）区域背景之间交替，创造电影节奏
- CTA 链接使用 980px 圆角——标志性的链接形状
- 图片始终置于纯色背景上，无竞争视觉元素
- 标题行高压缩到 1.07-1.14——标题以紧凑著称

### 不该做

- 不要引入额外的 accent 色——整个 chromatic 预算都花在蓝色上
- 不要使用重阴影或多个阴影层——只有一个柔和的漫射阴影或什么都没有
- 不要在卡片或容器上使用边框——几乎从不用可见边框（特定按钮除外）
- 不要给字体应用宽字间距——它被设计为在每个尺寸都紧凑运行
- 不要使用 800 或 900 字重——最大 700（粗体），甚至这也很少
- 不要在背景上添加纹理、图案或渐变——仅纯色
- 不要居中对齐正文——正文左对齐；只有标题居中
- 不要在矩形元素上使用大于 12px 的圆角（980px 仅用于药丸形）

## 8. Agent 提示指南

### 快速颜色参考

- 主 CTA：科技蓝（`#0071e3`）
- 页面背景（浅色）：`#f5f5f7`
- 页面背景（深色）：`#000000`
- 标题文字（浅色）：`#1d1d1f`
- 标题文字（深色）：`#ffffff`
- 正文文字：浅色上 `rgba(0, 0, 0, 0.8)`，深色上 `#ffffff`
- 链接（浅色 bg）：`#0066cc`
- 链接（深色 bg）：`#2997ff`
- 卡片阴影：`rgba(0, 0, 0, 0.22) 3px 5px 30px 0px`

### 示例组件提示

- "创建一个黑色背景的 hero 区域。标题 35px Noto Sans SC 字重 600，行高 1.07，字间距 -0.28px，颜色白色。副标题 21px Noto Sans SC 字重 400，行高 1.19，颜色白色。两个药丸 CTA：'了解更多'（透明 bg，白色文字，1px solid 白色边框，980px 圆角）和'购买'（科技蓝 #0071e3 bg，白色文字，8px 圆角，8px 15px 内边距）。"
- "设计一个产品卡：#f5f5f7 背景，8px 圆角，无边框，无阴影。产品图片占卡片顶部 60%，纯色背景。标题 21px Noto Sans SC 字重 400，字间距 0.196px，行高 1.14。描述 16px Noto Sans SC 字重 400，颜色 rgba(0,0,0,0.8)。'了解更多'链接用 #0066cc，16px。"
- "创建交替区域布局：第一个区域黑色 bg，白色文字，居中产品图片，第二个区域 #f5f5f7 bg，#1d1d1f 文字。每个区域接近全宽度，35px 标题，下方两个药丸 CTA。"
- "设计'了解更多'链接：浅色 bg 上文字 #0066cc 或深色 bg 上 #2997ff，16px Noto Sans SC，悬停下划线。文本后包含右箭头字符（>）。作为独立 CTA 时，包裹在 980px 圆角容器中形成药丸形状。"

### 迭代指南

1. 每个交互元素都用科技蓝（`#0071e3`）——没有其他 accent 色
2. 区域背景交替：黑色用于沉浸时刻，`#f5f5f7` 用于信息时刻
3. 字体光学尺寸：Noto Sans SC 用于 20px+，Noto Sans SC 用于以下——永不混用
4. 全尺寸负字间距：35px 时 -0.28px，19px 时 -0.374px，16px 时 -0.224px，12px 时 -0.12px
5. 产品始终出现在纯色背景上——Hero模块中从不出现在渐变、纹理或生活方式背景上
6. 阴影很少且始终柔和：`3px 5px 30px 0.22 透明度` 或完全没有
7. 药丸 CTA 使用 980px 圆角——这创造了标志性的胶囊形状
