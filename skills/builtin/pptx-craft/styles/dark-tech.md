# 设计系统：深色科技风

## 1. 视觉主题与氛围

深色科技风呈现了一种高对比度、技术前沿的视觉体验，通过克制的设计语言传达了强大的计算能力。设计建立在纯粹的黑（`#000000`）白（`#ffffff`）基础之上，点缀以标志性的绿色色（`#76b900`）——这个颜色是如此特殊，以至于它本身就是品牌的指纹。这不是大自然的葱郁之绿，而是电光般的、偏柠檬色的绿，介于黄绿色和翠绿色之间，任何科技行业的人一眼就能识别。

定制的 Noto Sans SC 字体营造出一种干净、硬朗的排版风格。标题使用 35px 粗体，行距紧凑为 1.25，形成密集而有说服力的文本块。这种字体干净利落、务实、工程导向。正文使用 16px，阅读舒适但并不宽松，传达出屏幕空间被精心优化的感觉。

设计对绿色点缀的使用极其克制。`#76b900` 仅出现在边框（`2px solid #76b900`）、下划线和 CTA 按钮上——但从不用作背景或大面积表面。绿色是信号色，不是表面色。结合深邃的阴影系统（`rgba(0, 0, 0, 0.3) 0px 0px 5px`）和极简的圆角（1-2px），整体效果如同用像素渲染的精密工程硬件。

**关键特征：**

- 绿色（`#76b900`）作为纯点缀色——仅用于边框、下划线和高亮
- 黑色（`#000000`）主背景，深色区域配白色（`#ffffff`）文字
- Noto Sans SC 字体——干净、硬朗、工程感
- 紧凑的行高（标题 1.25）形成密集、有说服力的文本块
- 极简圆角（1-2px）——锐利、工程感的角落
- 绿色边框按钮（`2px solid #76b900`）作为主要交互模式
- Font Awesome 6 Pro/Sharp 图标系统，字重 900 用于锐利图标

## 2. 色彩调色板与角色

### 主品牌色

- **绿色**（`#76b900`）：标志性颜色——边框、下划线、CTA 描边、活跃指示器。从不用作大面积填充。
- **纯黑**（`#000000`）：主背景，浅色表面上的文字，主色调。
- **纯白**（`#ffffff`）：深色背景上的文字，浅色区域背景，卡片表面。

### 扩展调色板

- **浅绿**（`#bff230`）：明亮的石灰 accent，用于高亮和悬停状态。
- **橙 400**（`#df6500`）：暖色 accent，用于警报、特色徽章。
- **黄 300**（`#ef9100`）：次要暖色 accent，类别高亮。
- **黄 050**（`#feeeb2`）：浅色暖色表面，用于标注背景。

### 状态与语义色

- **红 500**（`#e52020`）：错误状态，关键警报。
- **红 800**（`#650b0b`）：深红色，用于严重警告背景。
- **绿 500**（`#3f8500`）：成功状态，正面指示器（比品牌绿更深）。
- **蓝 700**（`#0046a4`）：信息提示，链接悬停替代色。

### 装饰色

- **紫 800**（`#4d1368`）：深紫色，用于渐变末端、高级/AI 场景。
- **紫 100**（`#f9d4ff`）：浅紫色表面色调。
- **品红 700**（`#8c1c55`）：丰富的 accent，用于特别促销或精选内容。

### 中性色阶

- **灰 300**（`#a7a7a7`）：柔和文字，禁用标签。
- **灰 400**（`#898989`）：次要文字，元数据。
- **灰 500**（`#757575`）：三级文字，占位符。
- **灰色边框**（`#5e5e5e`）：微妙边框，分割线。
- **近黑**（`#1a1a1a`）：深色表面，卡片背景。

### 交互状态

- **链接默认（深色背景）**（`#ffffff`）：深色背景上的白色链接。
- **链接默认（浅色背景）**（`#000000`）：浅色背景上的黑色链接，带绿色下划线。
- **链接悬停**（`#3860be`）：所有链接变体的蓝色悬停偏移。
- **按钮悬停**（`#1eaedb`）：青色高亮，用于按钮悬停状态。
- **按钮激活**（`#007fff`）：亮蓝色，用于活跃/按下按钮状态。

### 阴影与深度

- **卡片阴影**（`rgba(0, 0, 0, 0.3) 0px 0px 5px 0px`）：用于卡片的微妙环境阴影。

## 3. 排版规则

### 字体系列

- **Noto Sans SC**：`Noto Sans SC`
- **图标字体**：`Font Awesome 6 Pro`（字重 900 用于实心图标，700 用于常规）

### 层级

| 角色         | 字体         | 大小             | 字重 | 行高         | 字间距  | 说明                                  |
| ------------ | ------------ | ---------------- | ---- | ------------ | ------- | ------------------------------------- |
| 展示标题     | Noto Sans SC | 35px (2.19rem)   | 700  | 1.25（紧凑） | normal  | Hero 区最大影响力标题                 |

| 正文大号     | Noto Sans SC | 19px (1.19rem)   | 700  | 1.67（宽松） | normal  | 强调正文，引导段落                    |

| 微型标签     | Noto Sans SC | 10px (0.63rem)   | 700  | 1.50         | normal  | `text-transform: uppercase`，微小徽章 |

### 原则

- **粗体作为默认风格**：大量使用字重 700 用于标题、按钮、链接和标签。400 字重仅保留给正文和描述——其他一切都是粗体，传达自信和权威。
- **标题紧凑，正文宽松**：标题行高始终 1.25（紧凑），正文放松到 1.50-1.67。这种对比在内容块顶部创造视觉密度，在段落中提供舒适的阅读体验。
- **标签使用大写**：标签使用 `text-transform: uppercase` 和字重 700，读起来像硬件规格标签。
- **无装饰性字间距**：字间距始终 normal，除了紧凑按钮（0.144px）。字体本身承载了工业风格特征，不需要额外修饰。

## 4. 组件样式

### 按钮

**主按钮（绿色边框）**

- 背景：`transparent`
- 文字：`#000000`
- 内边距：11px 13px
- 边框：`2px solid #76b900`
- 圆角：2px
- 字体：16px 字重 700
- 悬停：背景 `#1eaedb`，文字 `#ffffff`
- 激活：背景 `#007fff`，文字 `#ffffff`，边框 `1px solid #003eff`
- 聚焦：背景 `#1eaedb`，文字 `#ffffff`，轮廓 `#000000 solid 2px`
- 用途：主 CTA（"了解更多"、"探索解决方案"）

**次按钮（绿色边框细线）**

- 背景：transparent
- 边框：`1px solid #76b900`
- 圆角：2px
- 用途：次要操作，替代 CTA

**紧凑 / 行内**

- 字体：14.4px 字重 700
- 字间距：0.144px
- 行高：1.00
- 用途：行内 CTA

### 卡片与容器

- 背景：`#ffffff`（浅色）或 `#1a1a1a`（深色区域）
- 边框：无（干净边缘）或 `1px solid #5e5e5e`
- 圆角：2px
- 阴影：`rgba(0, 0, 0, 0.3) 0px 0px 5px 0px` 用于悬浮卡片
- 悬停：阴影加深
- 内边距：16-24px

### 链接

- **深色背景上**：`#ffffff`，无下划线，悬停变为 `#3860be`
- **浅色背景上**：`#000000` 或 `#1a1a1a`，下划线 `2px solid #76b900`，悬停变为 `#3860be`，下划线消失
- **绿色链接**：`#76b900`，悬停变为 `#3860be`
- **柔和链接**：`#666666`，悬停变为 `#3860be`

### 图片处理

- 产品/技术渲染图作为Hero图，常全宽展示
- 截图带有微妙阴影增加深度
- 深色 Hero 区域的绿色渐变叠加
- 圆形头像容器，50% 圆角

### 特色组件

**产品卡片**

- 干净的白色或深色卡片，极小圆角（2px）
- 标题带绿色 accent 边框或下划线
- 粗体标题 + 浅色描述文字模式
- 底部带绿色边框的 CTA

**技术规格表**

- 工业风格网格布局
- 交替行背景（微妙灰色偏移）
- 粗体标签，常规数值
- 关键指标用绿色高亮

## 5. 布局原则

### 间距系统

- 基本单位：8px
- 刻度：1px, 2px, 3px, 4px, 5px, 6px, 7px, 8px, 9px, 10px, 11px, 12px, 13px, 15px
- 主要内边距值：8px, 11px, 13px, 16px, 24px, 32px
- 区域间距：48-80px 垂直内边距

### 网格与容器

- 最大内容宽度：约 1200px（居中限制）
- Hero 区：全宽区域，文字居中限制
- 功能区：2-3 列网格用于产品卡片
- 文档/对比内容：单列布局

### 留白理念

- **有目的的密度**：使用比典型 SaaS 网站更紧凑的间距，反映了技术内容的密度。留白用于分隔概念，而非制造奢华空荡感。
- **区域节奏**：深色区域与白色区域交替，使用背景色（而不仅是间距）分隔内容块。
- **卡片密度**：卡片间距 16-20px，营造产品目录感而非画廊感。

### 圆角刻度

- 微型（1px）：行内元素，微小元素
- 标准（2px）：按钮、卡片、容器、输入框——几乎万物的默认值
- 圆形（50%）：头像图片，圆形标签指示器

## 6. 深度与层次

| 级别                | 处理方式                          | 用途                  |
| ------------------- | --------------------------------- | --------------------- |
| 平面（0 级）        | 无阴影                            | 页面背景，行内文字    |
| 微妙（1 级）        | `rgba(0,0,0,0.3) 0px 0px 5px 0px` | 标准卡片，弹窗        |
| 边框（1b 级）       | `1px solid #5e5e5e`               | 内容分割线，区域边框  |
| 绿色 accent（2 级） | `2px solid #76b900`               | 活跃元素，CTA，选中项 |

**阴影理念**：深度系统极简且实用。本质上只有一个阴影值——5px 环境模糊、30% 透明度——仅偶尔用于卡片。主要的深度信号不是阴影而是**色彩对比**：黑色背景旁边是白色区域，绿色边框在深色表面上。这创造了类似硬件的视觉分层，深度来自材质差异而非模拟光照。

### 装饰深度

- Hero内容背后的绿色渐变洗染
- 深色到更深的渐变（黑色到近黑）用于区域过渡
- 无毛玻璃效果或模糊效果——清晰优于氛围

## 7. Agent 提示指南

### 快速颜色参考

- 主 accent：绿色（`#76b900`）
- 深色背景：纯黑（`#000000`）
- 浅色背景：纯白（`#ffffff`）
- 标题文字（深色 bg）：白色（`#ffffff`）
- 标题文字（浅色 bg）：黑色（`#000000`）
- 正文文字（浅色 bg）：黑色（`#000000`）或近黑（`#1a1a1a`）
- 正文文字（深色 bg）：白色（`#ffffff`）或灰 300（`#a7a7a7`）
- 链接悬停：蓝（`#3860be`）
- 边框 accent：`2px solid #76b900`
- 按钮悬停：青（`#1eaedb`）

### 示例组件提示

- "创建一个黑色背景的 hero 区域。标题 35px Noto Sans SC字重 700，行高 1.25，颜色 #ffffff。副标题 19px 字重 400，行高 1.67，颜色 #a7a7a7。CTA 按钮：透明背景，2px solid #76b900 边框，2px 圆角，11px 13px 内边距，文字 #ffffff。悬停：背景 #1eaedb，文字白色。"
- "设计一个产品卡片：白色背景，2px 圆角，阴影 rgba(0,0,0,0.3) 0px 0px 5px。标题 19px Noto Sans SC字重 700，行高 1.25，颜色 #000000。正文 16px 字重 400，行高 1.67，颜色 #757575。标题绿色下划线：border-bottom 2px solid #76b900。"
- "创建深色功能区：#000000 背景。区域标签 16px 字重 700 大写，颜色 #76b900。标题 23px 字重 700，颜色 #ffffff。描述 16px 字重 400，颜色 #a7a7a7。三个产品卡片一行，20px 间距。"

### 迭代指南

1. 始终使用 `#76b900` 作为 accent，从不用作背景填充——它是信号色，用于边框、下划线和高亮
2. 按钮默认透明背景配绿色边框——仅在悬停/激活状态下出现填充背景
3. 字重 700 是所有交互和标题元素的主导风格；400 仅用于正文段落
4. 圆角为 2px——这种锐利、极简的圆角是工业美学的核心
5. 深色区域使用白色文字；浅色区域使用黑色文字——绿色 accent 在两种表面上效果一致
6. 链接悬停始终为 `#3860be`（蓝色），无论链接的默认颜色
7. 标题行高 1.25，正文 1.50-1.67——保持这种对比以维持视觉层级
8. 标签使用大写 16px 粗体——这种硬件标签式的排版是风格的核心
