### 页3：什么是机器学习
- 定义：机器学习是人工智能（AI）的核心分支，让计算机不需要被明确编程就能从数据中学习和改进
- 核心对比：传统编程 vs 机器学习
  - 传统编程：数据 + 规则 → 答案
  - 机器学习：数据 + 答案 → 规则（模型）

### 页4：核心思想与对比
- 核心思想：让计算机通过经验（数据）自动改善性能
- 表格对比：传统编程（程序员写逻辑、适合规则明确的问题） vs 机器学习（算法发现模式、适合规则复杂的问题）

### 页5：机器学习基本流程
- 六步流程图：收集数据 → 准备/清洗 → 选择模型 → 训练模型 → 评估模型 → 部署使用
- 强调：调参优化循环

### 页6：三大学习类型
- 监督学习（Supervised Learning）：有标签数据，如猫狗分类
- 无监督学习（Unsupervised Learning）：无标签数据，自动发现结构
- 强化学习（Reinforcement Learning）：试错 + 奖励信号，如AlphaGo

### 页7：日常应用场景
- 抖音推荐：预测你喜欢的内容
- 人脸解锁：识别面部特征
- 语音助手：语音转文字 + 意图理解
- 垃圾邮件过滤：判断邮件模式
- 天气/股票预测：从历史数据发现规律

### 页8：总结页
- 一句话总结：机器学习 = 数据 + 算法 → 让计算机自己学会做决策
